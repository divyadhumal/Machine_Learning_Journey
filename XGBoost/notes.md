
# 🚀 XGBoost Machine Learning Project

Learning and implementing powerful boosting algorithms using XGBoost for real-world datasets.

## *📌 About the Project*

This project focuses on understanding and applying XGBoost, one of the most efficient machine learning algorithms.

It includes:

Classification problem
Regression problem
Model evaluation
Visualization
Hyperparameter tuning

-------------------------------

## 📊 Datasets Used
🔹 Breast Cancer Dataset
Type: Classification
Goal: Predict whether the tumor is malignant or benign

🔹 California Housing Dataset
Type: Regression
Goal: Predict house prices

--------------------------------
