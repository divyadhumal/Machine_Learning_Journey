# 📊 Boundary Extraction using Logistic Function

## 📌 Overview
This project demonstrates how a **decision boundary** is formed using a simple logistic model.  
We simulate data using age and income, then visualize how a model separates two classes.

---

## 🎯 Objective
- Understand how decision boundaries work  
- Visualize classification using a linear equation  
- Learn how features like **age** and **income** influence predictions  

---

## 🧠 Concept Used
The model uses a linear equation:

logit(p) = b0 + b1 * age + b2 * income

The **decision boundary** is where:

logit(p) = 0

---

## ⚙️ How It Works
- Random data for **age** and **income** is generated  
- A linear equation is applied to compute logits  
- Data points are classified into two categories:
  - `0` → Class 0  
  - `1` → Class 1  
- A decision boundary line is plotted  

---

## 📈 Visualization
- Scatter plot shows two classes  
- Decision boundary line separates them  
- Helps understand how classification models work
