
# 📊 Support Vector Machine (SVM) with Kernel Trick

This project demonstrates the implementation of **Support Vector Machine (SVM)** using different kernels to handle both linear and non-linear data.

## 🚀 What I Learned

- Basics of Support Vector Machines
- Concept of **decision boundary**
- Difference between **linear and non-linear classification**
- Importance of **kernel trick**
- How kernels transform data into higher dimensions

## 🧠 Kernels Used

- Linear Kernel
- Polynomial Kernel
- RBF (Radial Basis Function) Kernel

## 📌 Key Concepts

- Margin maximization
- Support vectors
- Hyperplane separation
- Kernel transformation

## 🛠️ Tech Stack

- Python 🐍
- NumPy
- Matplotlib
- Scikit-learn

## 📊 Visualization

The project includes visualizations of:
- Decision boundaries
- Data points classification
- Effect of different kernels
